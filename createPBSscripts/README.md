# Summary
Quick scripts to auto-create and run PBS scripts for batch processing

```createPBSAutomated.sh```Creates PBS script for batch processing using ```automatedDTIProcessingPipeline.sh``` script

```createPBSBedpostX.sh``` Creates PBS script for batch processing using ```preprocessBedpostData.sh``` script

```createPBSProbtrackx.sh``` Creates PBS script for batch processing using ```tractography.sh``` script
